## 참고
- 개별 항목 조회의 경우 여러 path가 등장 할 수 있기 때문에 기능에 따라 전부 작성해주세요.
    - ex) 친구를 id, name, class에 따라 개별 조회 할 경우
        
    - `/friends/:id` 
    - `/friends?name=name`
    - `/friends?class=class`
